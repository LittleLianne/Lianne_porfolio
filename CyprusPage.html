The following is a summary of the finding and recommendations for a website.

This was done using heuristic evaluation, this type of evaluation requires no users to be present, it uses the knowledge of typical users and is guided by guidelines, standards, and intuitive judgements. For this particular evaluation, I followed the usability inspection technique guided by Nielsen's list of heuristics.

Overall: The website’s usability and design is good; it uses a minimalist design with all the relevant information being easy to find through the site’s navigation.

The points below are recommendations for changes to the website:

  1. Aprons appear first on the accessories page, however, there is a button for aprons in the top nav bar, maybe deleting the button from the top menu for aprons would be better, than creating a side menu on the accessories page such as the following suggestions: -

    2. This first suggestion uses the wording view all products which can be clicked on to bring up a list of options, however, the user can select any of the images to go to the section they wish to view with the side menu still visible within each section making it easy to view whichever section they wish to look it.

    Both these examples would allow for more flexibility for the users and help them narrow down their search, which could lead to more products being purchased.

  2. When logging in if details are entered incorrectly, the warning words used are “invalid login credentials”, simpler words could be used such as incorrect logging-in, which would explain the error more to the user.

  3. The button for sign-in and the button for Join load the same page, the sign-in page could be simplified by just including the sign-in options and a small button which would take the user to register if they are not already registered, as shown below in the example.

  4. The GBP drop-down menu looks blank at first (1), but when rolled over USD appears (2), then EUR (3), all the words should be visible once GBP has been clicked.

  5. The wording cart is usually used on .com website, usually, on co.uk website the wording basket is used, as this word is more familiar to the user.
  6. The arrows on the main slide show on the home page do not work.
  7. The Facebook, Instagram, Twitter and Pinterest icons in the top left-hand corner move a little bit each time you move on to a different page within the website.
