4.Create all the materials needed for evaluation.

The following stages 5 to 11 were created.

5.Interface Sketch.

An interface sketch was created as this is a cheap and easy way to create a low-fidelity prototype.

6.Card based prototypes.

Another low-fidelity prototype that allows the participants of the evaluation to see the type of interaction that would be available on the smart plate.

7.Script.

The script was created to ensure all the participants were told the same information.

8.Questionnaire.

Questionnaires were created with mostly open structured questions which allowed for a more open answer than yes and no.

9.Consent form.

The consent form provides a brief outline to the participants on what will happen during the evaluation as well as deals with any privacy issues.

10.Task Cards.

Task cards were cards that have different tasks on them for the participants to do.

11.Data collections forms.

These forms were a great medium to allow me to write any remarks or comments the participants made as well as take notes if the participant was struggling to do a task for example.

12.The evaluation.

The evaluation allowed me to check that users could interact with the interface sketch and to see if it meets their needs and requirements.

13.Analysis the data.

Overall, three recommendations for changes to address were found after analysing the data.

14.Heuristic Evaluation.

Heuristic Evaluation was done which uses the knowledge of typical users and is guided by guidelines, standards, and intuitive judgement.
